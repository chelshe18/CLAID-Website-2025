# CLAID Website 2025

Official website for the Chinese Language Association for International Development (CLAID), bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Local Development

### Prerequisites

- Node.js 18+ recommended
- npm 9+ recommended

### Install

In the project directory, you can run:

```bash
npm install
```

### Run locally

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

## NPM Scripts

- `npm start`: starts the development server
- `npm test`: runs tests in watch mode
- `npm run build`: creates a production build in `build/`
- `npm run predeploy`: runs the production build before deploy
- `npm run deploy`: deploys `build/` using `gh-pages`

Note: `npm run dev` is defined in `package.json` but is not a standard Create React App command.

## Project Structure

```text
src/
	components/      Reusable UI pieces (navbar, footer, cards)
	pages/           Route-level pages (Home, About, Events, etc.)
	data/            JSON/CSV content used by pages
	scripts/         Data transformation scripts for board/member data
	assets/          Local image/media assets
public/
	images/          Public image files referenced by pages/data
```

## Routes

Defined in `src/App.js`:

- `/` Home
- `/about`
- `/connect`
- `/events`
- `/performance`
- `/tutoring`

## Content and Data Workflow

Board member content is managed from files in `src/data/` and generated by scripts in `src/scripts/`.

### Input files

- `src/data/raw_board.csv`: source board data

This data is tab-seperated, and will appear on the `About Us` page in the order of the rows in this file.

### Generated/intermediate files

- `src/data/board.csv`
- `src/data/board.json`
- `src/data/exec.json`
- `src/data/events.json`
- `src/data/performance.json`
- `src/data/tutoring.json`

### Script pipeline

To update board members, run the full board pipeline with:

```bash
node src/scripts/parse-board.js
```

This executes:

1. `convert-board.js`
2. `make-board.js`
3. `separate-board.js`

## Deployment

The project is configured for static deployment with:

- `homepage` set to `https://claid.org`
- `gh-pages` deploy command targeting `build/`

Typical deploy flow:

```bash
npm run predeploy
npm run deploy
```

## Notes

- Public static images are under `public/images/`.
- Route/page implementations are in `src/pages/`.
- Navigation and layout shell are in `src/components/` and `src/App.js`.
